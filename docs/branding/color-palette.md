# 颜色规范（Color Palette）

核心色
- Primary（品牌主色）
  - Hex: `#4C6FFF`（Indigo 500）
  - 用途：按钮、强调、链接、图标描边
- Accent（强调辅色）
  - Hex: `#00D1B2`（Teal / Mint）
  - 用途：高亮、渐变搭配、成功态辅助

语义色
- Success: `#10B981`（绿）
- Warning: `#F59E0B`（橙）
- Danger:  `#EF4444`（红）
- Info:    `#3B82F6`（蓝）

中性色（浅色模式优先）
- Text Primary: `#0F172A`（深蓝灰）
- Text Secondary: `#334155`
- Border: `#E2E8F0`
- Background: `#FFFFFF`
- Surface: `#F8FAFC`

深色模式建议
- Text Primary: `#E2E8F0`
- Text Secondary: `#CBD5E1`
- Border: `#1F2937`
- Background: `#0B1220`
- Surface: `#111827`

渐变（建议）
- 方向：左上 → 右下
- Stops: `#4C6FFF` → `#00D1B2`

对比度建议
- 文字/背景对比度 ≥ 4.5:1
- 次要文字/禁用态可放宽至 3:1，但不用于关键信息
